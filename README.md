# Tronsconeuse

un petit découpeur de fichiers bien sympa

# Compilation (requier Qt5)

1. qmake
2. make all clean
3. ./tronsconeuse

## Commentaire

je me develope ce petit outil car je ne trouve aucun dépoupeur de fichiers convivial et qui ne plante pas pour un oui ou pour un non, il execute le découpage/recolage sur un thread séparé
